\# Gesture Controlled Steering Wheel



A real-time gesture controlled racing wheel built using Python, OpenCV and MediaPipe.



\## Features



\- Two-hand steering control

\- Gesture-based acceleration and braking

\- Browser racing game compatibility

\- Adjustable steering sensitivity

\- Real-time hand tracking



\## Technologies Used



\- Python

\- OpenCV

\- MediaPipe

\- PyDirectInput



\## Installation



```bash

pip install -r requirements.txt

```



\## Run



```bash

python hand\_tracking.py

```



\## Controls



| Gesture | Action |

|----------|--------|

| Hands level | Drive straight |

| Left hand lower | Turn left |

| Right hand lower | Turn right |

| Hands apart | Accelerate |

| Hands close together | Brake |
